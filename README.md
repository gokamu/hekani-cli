# Usage
<!-- usage -->
```sh-session
$ npm install -g hekani-cli
$ hekani-cli COMMAND
running command...
$ hekani-cli (-v|--version|version)
hekani-cli/0.0.1 darwin-x64 node-v14.16.1
$ hekani-cli --help [COMMAND]
USAGE
  $ hekani-cli COMMAND
...
```
